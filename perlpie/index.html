Perl Pie : Replace String in Files
This will search all text files in the current directory for the word "SOME_NAME" and replace it with my name "Ger". This regex ends with "/gi" which the "i" means ignore case, so it will change any spelling of the word "SOME_WORD" have it be upper or lower case spelling of it.
Note also that you can just type in the full file name istead of "*.txt" and do the one single file.
perl -p -i -e 's/SOME_NAME/Ger/gi' ./*.txt
Perl Pie : Replace String in Named Files
This is the same as above only diffrence is it will change it only in files with the name of "filename"
perl -p -i -e 's/foo/bar' `find . -name "filename"` ./*.txt
