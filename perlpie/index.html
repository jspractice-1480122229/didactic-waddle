Perl Pie : Replace String in Files
perl -p -i -e 's/SOME_NAME/Ger/gi' ./*.txt
This will search all text files in the current directory for the word "SOME_NAME" and replace it with my name, "Ger". This regex ends with "/gi", in which the "i" means 'ignore case', so it will change any spelling of "SOME_NAME", have it be the upper or lower case spelling of it e.g. "Some_Name", "Some_NAME", "SoMe_NaME" and the like.
Note also that you can just type in the full file name istead of "*.txt" and do the one single file.
Perl Pie : Replace String in Named Files, recursively
perl -p -i -e 's/foo/bar' `find . -name "filename"`
This is the same as above, only diffrence is it will change it only in files with the name of "filename" in the current and any subdirectories so if you have a structure like
./filename
./bkp/filename
./original/filename
 it will overwrite them all.
Perl Pie : Make a backup with '-i.bkp' (or any extension you choose).
perl -p -i.bkp -e 's/SOME_NAME/Ger/gi' ./*.txt
perl -p -i.bkp -e 's/foo/bar' `find . -name "filename"`
This is safer than either method above, as a backup will be made when the changes to all those *.txt and "filename" are changed recursively, like so:
./myname.txt
./myname.txt.bkp
./theFavorite.txt
./theFavorite.txt.bkp
./filename
./filename.bkp
./bkp/filename
./bkp/filename.bkp
./original/filename
./original/filename.bkp
 Please note that this is a silly scenario but, if you are a beginner, safety first!
